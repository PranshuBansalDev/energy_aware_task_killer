Energy-Aware Task Killer

EE202B - Energy-Aware Priority Based Task Scheduler App for Android

Core Idea

Have you ever been out exercising with a health monitoring app, only to find that your health monitoring app has died?

Wish there was a way to make sure that your apps stayed alive, even when your battery is low?

We have the answer for you!

This app aims to allow users to select which apps run, and which apps to kill when your battery level falls below specified thresholds.

As this app was made for a classroom project, we take no responsibility for what this code does or how it is used.

We built and tested this app using Android Studio and a Nexus phone running Android v5.1.1.

The smart phone must be rooted for this app to function.
