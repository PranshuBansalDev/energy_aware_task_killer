System Requirements

  • Nexus 4.
  • Android v5.1.1.
  • Rooted device.

This code may work on other devices with other configurations, but we have only tested it on a rooted Nexus 4 device running stock Android v5.1.1 (no modifications to kernel).

Our system used TWRP (https://twrp.me/) and SuperUser (https://play.google.com/store/apps/details?id=eu.chainfire.supersu&hl=en)

Files

apk: compiled android application

shell: shell script to kill tasks

src: java source files

Energy-Aware Task Killer - Download (github)github.com

Installation Procedure

  1. Compile APK from source or obtain from GitHub (above).
  2. Install "BusyBox" onto device (for "awk" command in shell script) (right).
  3. Upload file 202bShellScript.sh to Android device at location /sdcard/Download/
  4. Run compiled APK on Android device.
